# templates/

Pages HTML Jinja2 de l'interface web.

| Template | Page | Description |
|---|---|---|
| `base.html` | Layout | Sidebar, thème sombre/clair, Plotly.js |
| `index.html` | Accueil | Tableau de bord, pipeline en 3 étapes |
| `annotation.html` | Annotation | Canvas polygones, zoom, plein écran |
| `prediction.html` | Prédiction | Résultats hiérarchiques, barres de score |
| `clustering.html` | Clustering | Scatter UMAP, métriques, méthode du coude |
| `labeling.html` | Étiquetage | Labels globaux par image |
| `corrections.html` | Corrections | Modification en masse, historique |
| `regions.html` | Régions | Superpixels, contours, anomalies |
| `documentation.html` | Documentation | Markdown rendu en HTML (3 onglets) |

Chaque page contient un tutoriel pliable (📘) qui explique quoi, pourquoi et comment.
